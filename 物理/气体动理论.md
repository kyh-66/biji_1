

### 一、 理想气体的状态方程与微观模型

这部分建立了宏观参量（压强 $p$、体积 $V$、温度 $T$）与微观参量（分子质量 $m$、分子数密度 $n$、速率 $v$）之间的初步联系。

- **理想气体状态方程（宏观）：**
    
    $$pV = \nu RT = \frac{M}{\mu} RT$$
    
    式中 $\nu$ 为物质的量，$R$ 为普适气体常量（$8.31 \text{ J/(mol}\cdot\text{K)}$），$M$ 为气体总质量，$\mu$ 为摩尔质量。
    
- **理想气体状态方程（微观）：**
    
    $$p = nkT$$
    
    式中 $n$ 为分子数密度（单位体积内的分子数），$k$ 为玻尔兹曼常量（$1.38 \times 10^{-23} \text{ J/K}$），且 $R = N_A k$。
    
- **理想气体的微观假设：**
    
    1. 分子本身的大小可以忽略不计（质点）。
        
    2. 分子之间除碰撞瞬间外，无相互作用力（无分子势能）。
        
    3. 分子之间的碰撞是完全弹性碰撞（不损失动能）。
        

### 二、 压强与温度的微观本质（核心推导）

这是气体动理论的灵魂，揭示了我们感受到的“压强”和“温度”到底是什么。

- **气体压强公式：** 压强是大量气体分子频繁碰撞器壁产生的平均效果。
    
    $$p = \frac{2}{3} n \bar{\varepsilon}_k = \frac{1}{3} n m \bar{v^2}$$
    
    式中 $\bar{\varepsilon}_k$ 为分子平均平动动能，$\bar{v^2}$ 为分子方均速度。
    
- **温度的微观本质：** 温度是大量气体分子热运动剧烈程度的标志。
    
    $$\bar{\varepsilon}_k = \frac{3}{2} kT$$
    
    **注意：** 温度只与分子的**平均平动动能**有关，与气体的种类、质量无关。
    

### 三、 能量按自由度均分定理与内能

气体分子不仅会平动，还会转动和振动。我们需要用“自由度”来分配能量。

- **自由度 ($i$)：** 确定一个分子在空间的位置和姿态所需的独立坐标数。
    
    - **单原子分子**（如氦、氖）：只有平动，**$i = 3$**。
        
    - **刚性双原子分子**（如氧气、氮气）：平动加两个方向的转动，**$i = 5$**。
        
    - **刚性多原子分子**（如水蒸气、二氧化碳）：平动加三个方向的转动，**$i = 6$**。
        
- **能量均分定理：** 在温度为 $T$ 的平衡态下，物质分子的每一个自由度都具有相同的平均动能，其大小为 $\frac{1}{2} kT$。
    
    一个分子的总平均动能为：$\bar{\varepsilon} = \frac{i}{2} kT$
    
- **理想气体的内能 ($E$)：**
    
    理想气体没有分子势能，其内能就是所有分子的动能之和。
    
    对于 $\nu$ 摩尔的理想气体，总内能为：
    
    $$E = \frac{i}{2} \nu RT$$
    
    **核心考点：** 理想气体的内能仅仅是温度 $T$ 的单值函数（状态量）。
    

### 四、 麦克斯韦速率分布律

气体中各个分子的速率大小不一，但大量分子的速率分布遵循一定的统计规律。

- **速率分布函数 $f(v)$：** 表示速率在 $v$ 附近单位速率区间内的分子数占总分子数的百分比。物理意义是：曲线下的面积代表概率。
    
    $$\int_{0}^{\infty} f(v) dv = 1 \quad \text{(归一化条件)}$$
    
- **三种具有统计意义的速率（极其重要，常考排序与计算）：**
    

|**速率名称**|**公式**|**物理意义**|**大小关系**|
|---|---|---|---|
|**最概然速率 ($v_p$)**|$v_p = \sqrt{\frac{2RT}{\mu}}$|分布曲线的峰值对应速率（拥有该速率附近的分子数最多）|最小|
|**平均速率 ($\bar{v}$)**|$\bar{v} = \sqrt{\frac{8RT}{\pi\mu}}$|所有分子速率的算术平均值（常用于计算碰撞频率）|居中|
|**方均根速率 ($v_{rms}$)**|$v_{rms} = \sqrt{\frac{3RT}{\mu}}$|速率平方的平均值的平方根（与分子平均平动动能直接相关）|最大|

**比例关系：** $v_p : \bar{v} : v_{rms} = 1 : 1.128 : 1.224$。

### 五、 气体分子的碰撞与平均自由程

由于分子非常密集，它们在运动中会不断相互碰撞。

- **平均碰撞频率 ($\bar{Z}$)：** 一个分子在单位时间内与其他分子碰撞的平均次数。
    
    $$\bar{Z} = \sqrt{2} \pi d^2 n \bar{v}$$
    
    式中 $d$ 为分子有效直径（或碰撞截面直径）。
    
- **平均自由程 ($\bar{\lambda}$)：** 一个分子在连续两次碰撞之间自由飞行的平均距离。
    
    $$\bar{\lambda} = \frac{\bar{v}}{\bar{Z}} = \frac{1}{\sqrt{2} \pi d^2 n}$$
    
    由于 $p = nkT$，代入后可得另一常用形式：
    
    $$\bar{\lambda} = \frac{kT}{\sqrt{2} \pi d^2 p}$$
    
    **结论：** 温度一定时，压强越低（真空度越高），平均自由程越大。
